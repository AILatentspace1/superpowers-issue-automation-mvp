# Superpowers Labels

| Label | Meaning | Triggered by |
| --- | --- | --- |
| `sp:needs-design` | Issue needs planning/design | issue opened |
| `sp:design-approved` | Human approved design output | human label |
| `sp:needs-plan` | Issue needs implementation plan | workflow |
| `sp:plan-approved` | Human approved implementation plan | human label |
| `sp:needs-implementation` | Implementation can start | workflow |
| `sp:implementation-ready` | Implementation is ready for test review | human / automation |
| `sp:needs-debug` | PR/check failure needs root-cause debug | workflow / human label |
| `sp:needs-code-review` | PR needs Superpowers review | workflow / human label |
| `sp:needs-release-check` | PR needs final verification | workflow / human label |
| `sp:blocked` | Human intervention required | human / automation |
| `sp:done` | Flow complete | human / automation |
